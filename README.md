# alpha-apparel
AI-powered investor apparel platform. Generate personalized finance-themed merchandise from any stock ticker.
## Daily Startup

Open PowerShell in the project root.

Activate the virtual environment:

```powershell
.\.venv\Scripts\Activate.ps1
```

Verify the environment:

```powershell
python --version
```

Expected output:

```
Python 3.13.x
```

You're now ready to work on Alpha Apparel.
```
Launching FastAPI:
```
from the backend folder:

 uvicorn app.main:app --reload
 ```
